# NameGator

> Front Web do projeto NameGator, baseado no projeto de mesmo nome do Rodrigo Branas

## Tecnologias

- VueJS
- VueTifyJS
- Axios

### Clonar/Configurar

`$ git clone https://github.com/PaulloClara/name-gator.git`\
`$ cd name-gator`\
`$ yarn install`

### Rodar em Dev/Hot-Reload

`$ yarn dev`

### Compilar Codigo

`$ yarn build`
